
 When the authorization is failed
1.Go to the Terminal of the Visual studio and Enter the command 'sfdx force:auth:device:login' .
2. Enter ABCDD user code in the verification URL https://test.salesforce.com/setup/connect . ( enter the autogenerated code on your screen ).
3. Login successful for username@sandbox.com. You can now close the browser.
OR:
 Go to sfdx-package.json file and change the port number
  "name": "App",
  "namespace": "",
  "sfdcLoginUrl": "https://login.salesforce.com",
  "sourceApiVersion": "55.0",
  "oauthLocalPort" : 7717
----------------------------INSTALLATION OF SFDX LOCAL SERVER-----------------------------------
1. Delete yarn.lock file in sfdx folder if any
2. npm install --global windows-build-tools
2. Move python to C drive
3. npm install -g node-gyp
4. sfdx plugins:install @salesforce/lwc-dev-server
5. sfdx plugins:update


---------------------------To retrieve a particular lwc component---------------------------------
sfdx force:source:retrieve -m LightningComponentBundle:helloWorld
